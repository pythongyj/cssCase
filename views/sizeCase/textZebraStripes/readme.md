## 一、文本行的斑马条纹的原理 -- 渐变背景色

> 给整个元素设置一个渐变背景色，设置渐变背景色的大小设置为渐变色条数乘字体行高，字体行高可以使用**em**单位（渐变色条数\*行高\*1em）（字体行的竖直方向，另一个方向设置为 auto）。设置渐变色为位置为 background-origin: content-box;
